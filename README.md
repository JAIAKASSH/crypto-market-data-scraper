# Crypto Market Data Scraper

A Python-based Selenium project that extracts real-time cryptocurrency data from CoinMarketCap.

## Features
- Extracts Top 10 cryptocurrencies
- Collects Price, Market Cap, and 24h Volume
- Converts price to INR
- Exports data to CSV
- Supports headless execution

## Tech Stack
- Python
- Selenium
- Pandas
- WebDriver Manager

## Output
Structured CSV file compatible with Excel and BI tools.
